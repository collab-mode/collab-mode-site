Tutorial

Basic Usage

  1. Open a new file that you want to collaborativly edit.
  2. Use M-x collab-mode to start collab-mode.
  3. Use M-x collab-login to login using your gmail account.
  4. Use M-x infinote-find-file to store the file and startthe ability to collaborate.

Optional usage

  1. M-x collab-list-users will show a list of your online friends from your gmail account
  2. M-x collab-chat will allow you to chat with other's that are editing the document.
    3. You can also chat with your online gmail friends by typing their gmail name then : then message. Example to chat with collab-mode@gmail.com you would type - collab-mode: hello collab-mode
