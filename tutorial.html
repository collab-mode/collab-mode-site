Installation

There are two ways to install collab-mode

el-get (recommended)

  1. If you don't have it already, install el-get.
  2. Download collab-mode.rcp and save it into your recipes folder. That will likely be ~/.emacs.d/el-get/el-get.
  3. Install via M-x el-get-install then collab-mode. With el-get, you can also update by using M-x el-get-update then collab-mode.

manual install

  1. Download the latest version of collab-mode from GitHub.
  2. Add the directory where you downloaded collab-mode to your Emacs load-path.
  3. Add a line to require collab-mode to your Emacs init file.
