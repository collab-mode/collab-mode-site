Tutorial

Installation

There are two ways to install collab-mode

el-get (recommended)

  1. If you don't have it already, install el-get.
  2. Download collab-mode.rcp and save it into your recipes folder. That will likely be ~/.emacs.d/el-get/el-get.
  3. Install via M-x el-get-install then collab-mode. With el-get, you can also update by using M-x el-get-update then collab-mode.

manual install

  1. Download the latest version of collab-mode from GitHub.
  2. Add the directory where you downloaded collab-mode to your Emacs load-path.
  3. Add a line to require collab-mode to your Emacs init file.

Basic Usage

  1. Open a new file that you want to collaborativly edit.
  2. Use M-x collab-mode to start collab-mode.
  3. Use M-x collab-login to login using your gmail account.
  4. Use M-x infinote-find-file to store the file and startthe ability to collaborate.

Optional usage

  1. M-x collab-list-users will show a list of your online friends from your gmail account
  2. M-x collab-chat will allow you to chat with other's that are editing the document. You can also chat with your online gmail friends by typing their gmail name then : then message. Example to chat with collab-mode@gmail.com you would type - collab-mode: hello collab-mode
